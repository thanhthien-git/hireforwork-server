# hireforwork-server
SERVICE: do business logic ( crud, etc )
HANDLER: handling request from api call
ROUTER: define the url which is use for api call
# how to run
Add .env file
-> go run main.go
# how to create a api call
Download Postman
Enter localhost:8080 + router ( define in router.go )
